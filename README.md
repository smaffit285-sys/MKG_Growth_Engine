# SMWS Website

Static launch site for Sean Maffit Web Systems.

Production domain: `smws.work`

## Deployment

This repo is prepared as a static site. The public entry point is `index.html`; supporting pages are plain HTML with shared `styles.css`.

Required deployment settings:

- Framework preset: Other / Static
- Build command: none
- Output directory: repository root
- Production domain: `smws.work`

## Notes

This repository is prepared for SMWS launch use only. Operational apps and client systems belong in separate repositories.
